# Leit Gallery

> A static showcase of famous repository musical signatures, generated by [Leit](https://github.com/clement-joye/Leit).

## What is this?

This is a public gallery that demonstrates Leit's capabilities by showcasing pre-computed musical signatures for well-known open-source repositories. Each repository's architecture, complexity, and code identity is transformed into a unique, deterministic piece of music.

## Live site

🎵 **[View the Gallery](https://clement-joye.github.io/Leit-Gallery/)**

## Architecture

```
leit-gallery/
├── src/
│   ├── pages/           # Astro pages (index + /gallery/[slug])
│   ├── components/      # UI components (cards, audio player, metrics)
│   ├── layouts/         # Base HTML layout
│   ├── styles/          # Global CSS (Tailwind, dark-violet theme)
│   └── data/            # Gallery entries JSON + TypeScript types
├── public/
│   ├── badges/          # Static SVG badges per repo
│   └── images/          # Repo logos
├── scripts/
│   ├── seed_gallery.py  # Run Leit on curated repos + upload to Azure
│   └── generate_badges.py  # Generate SVG badges from gallery data
└── .github/workflows/
    └── deploy.yml       # GitHub Pages deployment
```

## How it works

1. **Generation** (one-time, local): Run `scripts/seed_gallery.py` to analyze curated repos via the Leit CLI. This produces WAV + MIDI files uploaded to Azure Blob Storage, and updates `src/data/gallery.json`.

2. **AI audio** (optional): Feed the Leit-generated MIDI/prompt to your preferred AI music model (Gemini, Suno, etc.) and upload the result to Azure Storage.

3. **Build & Deploy**: Push to `main` → GitHub Actions builds the Astro static site → deployed to GitHub Pages.

4. **Audio hosting**: All audio files (WAV, MIDI, AI-generated) are served from Azure Blob Storage (public read container, negligible cost for static files).

## Local development

```bash
npm install
npm run dev
```

## Adding a new repository

1. Add the repo to `CURATED_REPOS` in `scripts/seed_gallery.py`
2. Run: `python scripts/seed_gallery.py --repos your-slug --upload`
3. Add the entry to `src/data/gallery.json`
4. (Optional) Add a logo to `public/images/`
5. Run: `python scripts/generate_badges.py`
6. Commit and push
